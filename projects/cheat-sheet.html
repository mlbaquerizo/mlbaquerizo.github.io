Ruby Cheat Sheet

Hashes

Example: 

  hash = {"key1" => value1,
          "key2" => value2
          }
  • A hash is a dictionary of key, value pairs
  • Can be used to store values of properties, or named parameters
  • Keys and values can be any kind of Ruby object
  • Keys are often stored as symbols:
    • 
  hash = {key1: value1,
          key2: value2
          }
  • Access values:
    • 
  hash[:key1]
  => value1
  • Assign new values:
    • 
  hash[:key3] = value3
  hash[:key3]
  => value3
  • Useful Hash methods: each, each_key,each_value

Arrays

Example: 

  array = [thing1, thing2, thing3]
  • An array is an ordered list of values
  • Can be used to store values without properties
  • Indexed, each value has an index in the Array
  • Indices start at the first object with index 0; proceed in sequential order
  • Similar to hashes, where the keys are numbers from 0..(hash length)
  • Values can be any kind of Ruby object
  • Access values in array:
    • 
  array[0]
  => thing1
  array[1]
  => thing2
  • Assign new values:
    • 
  array[3] = thing4
  => thing4
  array
  => [thing1, thing2, thing3, thing4]
  • Using push method to push new value to end of array:
    • 
  array.push(thing5)
  => [thing1, thing2, thing3, thing4, thing5]
  • Useful Array Methods: each, push, pop, insert, map, slice

Strings

Example: 

  string = "I am a string"
  • A string is a set of characters inside quotes
  • Used to store information understood by users
  • Useful for user interaction with programs
  • Access parts of string by index:
    • 
  string[0]
  => "I"
  string[0..3]
  => "I am"
  • Useful String Methods: +, *, upcase, downcase, capitalize, concat, insert, length, reverse, split, slice, to_i, to_f,to_sym

Integers

Example: 

  num1 = 6
  num2 = 3
  • Integers are negative and positive whole numbers, and 0
  • Useful for loops/iterators and storing numerical data about Hashes, Arrays, or strings.
  • Can be operated on using mathematical functions:
    • 
  num1 + num2
  => 9
  num1 - num2
  => 3
  num1 * num2
  => 18
  num1 / num2
  => 2
  num1 % num2  # returns remainder of quotient of two numbers
  => 0
  • Useful Integer Methods: +, -, *, /, %, to_s, odd?, even?, times{block},
